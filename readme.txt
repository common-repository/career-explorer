Job Search Plugin is designed to streamline and enhance your job hunting experience.


== Description ==

The Career Explorer Plugin is a powerful and user-friendly tool designed to streamline the job search experience for users on your website or platform.

Whether you run a career-oriented website, a corporate portal, or an online community, this plugin provides a seamless integration of job search functionality, empowering users to find relevant employment opportunities effortlessly.

Integrated seamlessly into various platforms and websites, this plugin provides users with a comprehensive set of features to simplify the job search process.




== Installation ==

- **Minimum Requirements**
	PHP 7.4 or greater is required (PHP 8.0 or greater is recommended)
	MySQL 5.6 or greater, OR MariaDB version 10.1 or greater, is required

- **Automatic installation**
	Automatic installation is the easiest option — WordPress will handle the file transfer, and you won’t need to leave your web browser. To do an automatic install of Career Explorer, log in to your WordPress dashboard, navigate to the Plugins menu, and click “Add New.”
	In the search field type “Career Explorer,” then click “Search Plugins.” Once you’ve found us, you can view details about it such as the point release, rating, and description. Most importantly of course, you can install it by! Click “Install Now,” and WordPress will take it from there.

- **Manual installation**
	Manual installation method requires downloading the Career Explorer plugin and uploading it to your web server via your favorite FTP application. The WordPress codex contains <a href="https://wordpress.org/documentation/article/manage-plugins/#manual-plugin-installation">instructions on how to do this here</a>.



=== Features ===

- **Unified Job Search** : Access a centralized job search platform that aggregates listings from multiple sources, saving you time and effort by presenting a diverse range of opportunities in one place.

- **Advanced Filters** : Refine your search with precision using advanced filters such as location, job type. Tailor your criteria to find the perfect job that matches your skills and preferences.

- **Real-time Updates** : Receive instant notifications about new job postings and updates in your preferred industry. Stay ahead of the competition by being among the first to apply to freshly listed opportunities.

- **Remote Job Filters** : Integrate filters specifically for remote job opportunities, catering to the growing demand for flexible work arrangements and remote-friendly positions.

- **Company Insights** : Gain valuable insights into prospective employers. Access company profiles, reviews, and relevant information to make informed decisions about the organizations you're interested in.

- **Customizable Search Filters** : Tailor the job search experience with customizable filters such as location, industry, job type, and keywords, allowing users to refine their search results.

- **Quick and Accurate Results** : Utilize advanced search algorithms to provide users with quick and accurate job matches based on their criteria.


## Third-Party Service Usage

This plugin utilizes the Careerjet API service to fetch job listings based on user-defined parameters. The Careerjet API allows us to provide up-to-date job listings to our users directly within the plugin interface.

### Careerjet API

- **Service Description**: The Careerjet API provides access to a vast database of job listings from various sources across the internet.
- **Usage in Plugin**: The Careerjet API is used to retrieve job listings based on user queries within the plugin's job listing functionality.
- **Service Link: [Careerjet API](http://public.api.careerjet.net/)
- **Terms of Use: [Careerjet API Terms of Use](http://public.api.careerjet.net/terms)
- **Privacy Policy: [Careerjet API Privacy Policy](http://public.api.careerjet.net/privacy)

By utilizing the Careerjet API, we aim to enhance the functionality of our plugin and provide valuable job search features to our users. We are committed to transparently disclosing our use of third-party services to ensure compliance with legal requirements and user trust.
